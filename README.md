# Proiect-PIU

_**Meniu cu 3 jocuri unice.**_

Vreau să fac un meniu unde am 3 opțiuni, și, eventual, o a patra pentru setării gen resolutie sau controluri. Cele 3 opțiuni inițiale vor fi Snake, X și 0 sau Connect 4. Toate jocurile trebuie sa fie playable și vor fi făcute în C#. Planul meu ar fii mai întăi să programez jocurile în VS să văd daca funcționează, și ulterior, daca un joc nu poate fii integrat să îl înlocuiesc cu altul. După ce verific funcționarea tuturor jocurilor voi face meniul și voi adaugă aceste jocuri că opțiuni.
